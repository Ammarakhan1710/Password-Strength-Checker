# Password-Strength-Checker

A Python tool to evaluate the strength of passwords and provide simple feedback.

## Project Overview
This project is a Python-based password strength checker that analyzes user-provided passwords and classifies them as Weak, Moderate, or Strong based on the following criteria:
- Minimum length of 8 characters
- Presence of uppercase letters
- Presence of lowercase letters
- Presence of numbers
- Presence of special characters

It is designed to help users improve their password security by providing clear classification.

## Features
- Easy-to-use Python command-line tool
- Uses regular expressions to check password patterns
- Provides quick feedback on password strength
- Can be extended to give detailed suggestions for improvement
